# Terminology Guide

This file records stable mappings between code-facing English names and book-facing Turkish terminology.

| Code term | Book-facing term |
|---|---|
| topology | topoloji |
| topological space | topolojik uzay |
| basis | taban |
| subbasis | alt taban |
| closure | kapanış |
| interior | iç |
| boundary | sınır |
| continuity | süreklilik |
| homeomorphism | özdeşbiçimlilik |
| separation axioms | ayırma aksiyomları |
| countability | sayılabilirlik |
| connectedness | bağlılık |
| compactness | kompaktlık |
| invariants | değişmezler |
| cardinal functions | kardinal fonksiyonlar |
| theorem engine | kuram motoru |
| capabilities | destek düzeyleri |

Code names remain English. Manuscript and explanatory material may use Turkish.

## v0.7.9 note

Faz J'in ilk gerçek uygulama turunda editöryal rol adları daha açık sabitlendi:

- `selected block`: ana metnin taşıdığı çekirdek kanıt akışını bozmadan saklanması gereken, fakat sonraki bölümlerde gerçekten yeniden kullanılacak seçilmiş sonuç ailesi

Bu sürümden sonra Cilt II Bölüm 29--31 ile Cilt III Bölüm 32--36 kapanış bloklarında bu üç editöryal rol aynı ritimle kullanılacaktır.
