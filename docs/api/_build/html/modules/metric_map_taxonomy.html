Metric Map Taxonomy

Profile-based taxonomy for maps between metric spaces.

The helpers in this module deliberately classify only what can be certified from explicit finite data, or what the caller records as symbolic metadata. They are not a general analysis proof engine for arbitrary functions.

Classify an explicit map between finite metric spaces.

Finite metric spaces are discrete, so every explicit map between them is continuous and uniformly continuous. Lipschitz, non-expansive, isometry, similarity, bijectivity, and homeomorphism are certified by enumeration.
