Separation Basic

Core separation-axiom engine — T0, T1, Hausdorff (T2), Urysohn (T2.5).

This module contains: - All shared data (TRUE_TAGS, FALSE_TAGS, SUPPORTED_PROPERTIES, …) - The central dispatcher analyze_separation() - All private finite and theorem-level helpers - Convenience wrappers for the basic separation levels

Advanced queries (T3–T6, Tychonoff multi-criterion, separation_chain) live in pytop.separation_advanced. The legacy entry point pytop.separation re-exports everything from both modules.
