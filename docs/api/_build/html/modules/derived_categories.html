Derived Categories

Derived categories: triangulated structure, t-structures, semiorthogonal decompositions, derived functors, dg-enhancements, and the geometry of D^b(Coh(X)).

Key theorems and constructions implemented

  • Derived category D(A) (Grothendieck-Verdier 1963): for an abelian category A, the derived category D(A) is the localization of the homotopy category K(A) at the multiplicative system of quasi-isomorphisms (morphisms inducing isomorphisms on all cohomology groups). Objects of D(A) are cochain complexes of objects in A; morphisms are equivalence classes of roofs A <- C -> B where C -> A is a quasi-isomorphism. Variants: D^+(A), D^-(A), D^b(A) for complexes bounded below, above, or on both sides. The natural functor A -> D(A) embeds A as the full subcategory of complexes concentrated in degree 0. Ext^n_A(M, N) = Hom_{D(A)}(M, N[n]) for all n >= 0.

  • Triangulated categories (Verdier 1963, Puppe 1962): an additive category T with an autoequivalence [1]: T -> T (the shift or translation functor, with inverse [-1]) and a class of distinguished triangles A -> B -> C -> A[1] satisfying: (TR1) Every morphism f: A -> B embeds in a distinguished triangle A -> B -> C -> A[1]. (TR2) Rotation: if A -> B -> C -> A[1] is distinguished, so is B -> C -> A[1] -> B[1]. (TR3) Morphism of triangles: any commuting square A -> B extends to a morphism of

    distinguished triangles (not necessarily uniquely).

    (TR4) Octahedral axiom: for A -> B -> C’ -> A[1] and B -> C -> A’ -> B[1], there exists

    a triangle C’ -> D -> A’ -> C’[1] compatible with both.

    The homotopy category K(A) and the derived category D(A) are the canonical examples. The long exact sequence in cohomology arises from a distinguished triangle: H^n(A) -> H^n(B) -> H^n(C) -> H^{n+1}(A) -> … for A -> B -> C -> A[1].

  • t-structures (Beilinson-Bernstein-Deligne 1982): a t-structure on a triangulated category T is a pair (T^{<=0}, T^{>=0}) of full subcategories satisfying: (i) T^{<=0}[1] ⊂ T^{<=0} and T^{>=0}[-1] ⊂ T^{>=0}. (ii) Hom(X, Y) = 0 for X ∈ T^{<=0} and Y ∈ T^{>=1} = T^{>=0}[-1]. (iii) For every A ∈ T there is a distinguished triangle tau^{<=0}A -> A -> tau^{>=1}A -> …

    (the truncation triangle).

    The heart C = T^{<=0} ∩ T^{>=0} is an abelian category; the cohomology objects H^n(A) = tau^{>=0}tau^{<=0}(A[-n]) ∈ C. The standard t-structure on D(A) has heart ≅ A. Perverse t-structure (BBD 1982): on D^b_c(X) for a stratified space X, the perverse t-structure has heart = perverse sheaves. The middle extension IC(L) of a local system L on a locally closed stratum is the perverse sheaf of intersection cohomology.

  • Derived functors (Grothendieck 1957, Verdier): for a left exact functor F: A -> B between abelian categories (with enough injectives in A), the right derived functors R^nF: A -> B are defined by R^nF(M) = H^n(F(I^bullet)) where I^bullet is an injective resolution of M. In derived category language: RF: D^+(A) -> D^+(B) is the total right derived functor, a triangulated functor satisfying RF(M) ≅ F(I^bullet) for any injective resolution. Similarly, LF: D^-(A) -> D^-(B) for the left derived functor of a right exact F. Key examples: RHom(M, -), - ⊗^L N (derived tensor product), Rf_*(F) (direct image), f^!(F) (exceptional inverse image), R Gamma(X, F) (derived global sections).

  • Semiorthogonal decomposition (Bondal 1990): a semiorthogonal decomposition of a triangulated category T is a sequence of full triangulated subcategories A_1,…,A_n such that: (i) Hom(A_i, A_j) = 0 for i > j (semiorthogonality); (ii) T is generated by A_1,…,A_n (every object admits a filtration with associated graded pieces in A_1,…,A_n via iterated distinguished triangles). Written T = ⟨A_1,…,A_n⟩. The categories A_i are called admissible subcategories (their inclusion functors have left and right adjoints). Exceptional collections: an object E in T is exceptional if Hom(E, E) = k (the base field) and Ext^i(E, E) = 0 for i ≠ 0. A sequence E_1,…,E_n is exceptional if each E_i is exceptional and Ext^i(E_p, E_q) = 0 for all i and p > q. The subcategory generated by E_1,…,E_n is then semiorthogonal. Beilinson’s collection (1978): D^b(Coh(P^n)) = ⟨O, O(1), …, O(n)⟩ — the line bundles O(0), O(1), …, O(n) form a full exceptional collection on P^n.

  • Fourier-Mukai transforms (Mukai 1981): for smooth proper varieties X, Y over k and an object P ∈ D^b(Coh(X × Y)) (the ‘kernel’), the Fourier-Mukai functor is Phi_P: D^b(Coh(X)) -> D^b(Coh(Y)), Phi_P(F) = Rp_{Y*}(P ⊗^L p_X^*(F)). Every exact functor between derived categories of smooth proper varieties is a Fourier-Mukai functor (Orlov representability theorem 1996). Fourier-Mukai equivalences (derived equivalences): Phi_P is an equivalence iff the kernel P satisfies certain spanning and orthogonality conditions (Bondal-Orlov criterion). Mukai’s original example: the Poincaré bundle on A × A^vee (abelian variety and its dual) gives an equivalence D^b(Coh(A)) ≅ D^b(Coh(A^vee)) — the ‘Fourier transform’ on A.

  • dg-enhancements (Bondal-Kapranov 1990): a dg-enhancement of a triangulated category T is a pre-triangulated dg-category A with H^0(A) ≅ T. Every algebraic triangulated category (triangulated category arising as the homotopy category of a stable model category) admits a dg-enhancement (uniqueness: Lunts-Orlov 2010 for D^b(Coh(X)) and K^b(A-mod) for smooth proper X). The dg-structure captures the higher Ext groups and the A∞-structure of compositions, making the category ‘linear’ over a dg-algebra.

  • Bondal-Orlov reconstruction (1997): a smooth projective variety X is uniquely determined (up to isomorphism) by its derived category D^b(Coh(X)) when the canonical bundle omega_X is ample or anti-ample. More precisely: if Phi_P: D^b(Coh(X)) -> D^b(Coh(Y)) is an equivalence and X has ample (anti-)canonical bundle, then X ≅ Y. For K3 surfaces and abelian varieties (where omega ≅ O), derived equivalence does NOT imply isomorphism (Mukai pairs: D^b(A) ≅ D^b(A^vee) for dual abelian varieties).

pytop.derived_categories.is_triangulated(space: Any) → Result

Check whether the category is triangulated.

A triangulated category has a shift functor [1] and distinguished triangles A -> B -> C -> A[1] satisfying axioms TR1-TR4 (rotation, mapping cone existence, morphism of triangles, octahedral axiom). Every derived category D(A) and homotopy category K(A) is triangulated.

Decision layers

  1. Explicit ‘triangulated_category’ or ‘distinguished_triangle’ tag -> true.

  2. Derived category or homotopy category tags -> true.

  3. Non-triangulated (plain abelian, plain additive) tags -> false.

  4. Unknown otherwise.

pytop.derived_categories.has_t_structure(space: Any) → Result

Check whether the triangulated category carries a t-structure.

A t-structure (BBD 1982) on a triangulated category T is a pair of full subcategories (T^{<=0}, T^{>=0}) with: (i) T^{<=0}[1] ⊂ T^{<=0}; (ii) Hom(T^{<=0}, T^{>=1}) = 0; (iii) every object fits in a truncation triangle tau^{<=0}A -> A -> tau^{>=1}A. The heart C = T^{<=0} ∩ T^{>=0} is an abelian category.

Decision layers

  1. Explicit ‘t_structure’ or ‘heart_of_t_structure’ tag -> true.

  2. Derived category tags (always have standard t-structure) -> true.

  3. Perverse sheaves tags -> true (perverse t-structure).

  4. Triangulated-only without t-structure tags -> unknown.

  5. Unknown otherwise.

pytop.derived_categories.has_semiorthogonal_decomposition(space: Any) → Result

Check whether the derived category admits a semiorthogonal decomposition.

A SOD T = ⟨A_1,…,A_n⟩ requires semiorthogonality (Hom(A_i, A_j) = 0 for i > j) and generation. Smooth varieties with ample or anti-ample canonical bundle often admit SODs via exceptional collections. Varieties with trivial canonical bundle (CY, K3, abelian) do NOT admit full exceptional collections (by Serre functor rigidity).

Decision layers

  1. Explicit ‘semiorthogonal_decomposition’ or ‘exceptional_collection’ tag -> true.

  2. Projective space / Beilinson collection tags -> true.

  3. Trivial canonical bundle (CY, K3, abelian variety) -> false (no full SOD).

  4. Unknown otherwise.

pytop.derived_categories.is_dg_enhanced(space: Any) → Result

Check whether the triangulated category admits a dg-enhancement.

A dg-enhancement is a pre-triangulated dg-category A with H^0(A) ≅ T. Every algebraic triangulated category (arising from a stable model category) admits a dg-enhancement. Lunts-Orlov (2010): D^b(Coh(X)) for smooth proper X has a unique dg-enhancement. The topological stable homotopy category SH does NOT admit a dg- enhancement (it is not algebraic in the sense of Schwede).

Decision layers

  1. Explicit ‘dg_enhancement’ or ‘dg_category’ tag -> true.

  2. Derived category of coherent sheaves or modules (algebraic) -> true.

  3. Topological stable homotopy category -> false.

  4. Unknown otherwise.

pytop.derived_categories.mapping_cone_complex(boundaries_a: list[list[list[int]]], boundaries_b: list[list[list[int]]], chain_map: list[list[list[int]]]) → list[list[list[int]]]

Mapping cone C(f) for a chain map f: A → B.

C(f)_n = A_{n-1} ⊕ B_n with boundary matrix:

∂^C_n = [[-∂^A_{n-1},   0   ],
         [ f_{n-1},   ∂^B_n  ]]

boundaries_a[k] is ∂^A_k : A_k → A_{k-1} (shape dim(A_{k-1}) × dim(A_k)). chain_map[k] is f_k : A_k → B_k.

Returns boundary matrices of C(f).

Examples

Empty inputs yield an empty cone:

>>> mapping_cone_complex([], [], [])
[]
pytop.derived_categories.derived_functor_h(boundaries: list[list[list[int]]], degree: int) → dict[str, Any]

H_{degree} of a chain complex given by its boundary matrices.

boundaries[k] is ∂_k : C_k → C_{k-1} (shape dim(C_{k-1}) × dim(C_k)).

Returns

dict with keys degree, betti_number, torsion_coefficients, rank_in, rank_out, kernel_dimension.

Examples

Single edge (interval): H_0 = ℤ, H_1 = 0:

>>> derived_functor_h([[[-1, 1]]], 0)["betti_number"]

>>> derived_functor_h([[[-1, 1]]], 1)["betti_number"]
0
pytop.derived_categories.triangulated_structure_check(boundaries_a: list[list[list[int]]], boundaries_b: list[list[list[int]]], boundaries_c: list[list[list[int]]], f_mats: list[list[list[int]]], g_mats: list[list[list[int]]]) → dict[str, Any]

Check necessary conditions for A -f-> B -g-> C to be a distinguished triangle.

Verifies g ∘ f = 0 in each degree (necessary for any triangle) and whether the mapping cone C(f) has the same Betti numbers as C.

Returns

dict with composition_zero, degree_checks, cone_betti, c_betti, cone_matches_c.

Examples

Zero chain map g trivially satisfies g ∘ f = 0:

>>> r = triangulated_structure_check([], [], [], [[[1]]], [[[0]]])
>>> r["composition_zero"]
True
