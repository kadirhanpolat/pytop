Simplicial Complexes

Finite simplicial complex helpers for the geometric topology bridge.

class pytop.simplicial_complexes.SimplicialComplex(simplexes: Iterable[Simplex | Iterable[Any]], *, metadata: dict[str, Any] | None = None, require_face_closed: bool = True)

Bases: object

A finite abstract simplicial complex.

The family stores nonempty simplexes only. The empty face is treated as a standard background convention and is not materialized as a Simplex.
