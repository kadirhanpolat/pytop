Finite Map Analysis

Core finite-map types, property analysis engine, and map builders.

This module contains: - FiniteMap — explicit finite map between topological spaces - The central dispatcher analyze_map_property() and property wrappers - Map-algebra helpers (preimage, image, restriction, fixed points, inverse) - Builder functions (make_function, identity_function, constant_function) - Taxonomy / reporting utilities - All private finite-analysis helpers

Pointwise continuity and criterion-based checks live in pytop.map_continuity. The legacy entry point pytop.maps re-exports everything from both modules.
