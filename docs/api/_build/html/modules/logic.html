Logic

Propositional logic and quantifier helpers for foundational mathematics.

Provides Proposition (a labeled truth value), connective operations (negate, conjunction, disjunction, implies, iff), and finite quantifiers (for_all, there_exists, unique_exists).

class pytop.logic.Proposition(name: str, truth_value: bool)

Bases: object

A named propositional statement with a concrete truth value.

Parameters

Human-readable label for the statement.

Boolean truth value of the statement.

pytop.logic.negate(p: Proposition) → Proposition

Return ¬p.

pytop.logic.conjunction(*props: Proposition) → Proposition

Return p₁ ∧ p₂ ∧ … (logical AND of all propositions).

pytop.logic.disjunction(*props: Proposition) → Proposition

Return p₁ ∨ p₂ ∨ … (logical OR of all propositions).

pytop.logic.implies(p: Proposition, q: Proposition) → Proposition

Return p → q (equivalent to ¬p ∨ q).

pytop.logic.iff(p: Proposition, q: Proposition) → Proposition

Return p ↔ q (equivalent to (p → q) ∧ (q → p)).

pytop.logic.for_all(carrier: Iterable[Any], predicate: Callable[[Any], bool]) → bool

Return True iff predicate holds for every element of carrier (∀).

Parameters

carrier:

Finite iterable of elements to quantify over.

predicate:

A callable that returns a bool for each element.

pytop.logic.there_exists(carrier: Iterable[Any], predicate: Callable[[Any], bool]) → bool

Return True iff predicate holds for at least one element of carrier (∃).

pytop.logic.unique_exists(carrier: Iterable[Any], predicate: Callable[[Any], bool]) → bool

Return True iff predicate holds for exactly one element of carrier (∃!).
