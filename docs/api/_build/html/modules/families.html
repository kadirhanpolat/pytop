Families

Families of sets, covers, refinements, and partitions.

This module provides a small explicit layer for the foundational chapters of the book ecosystem. The goal is not abstract generality but clear, finite, testable behavior for common classroom constructions.

pytop.families.normalize_family(family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]]) → tuple[frozenset[Any], ...]

Normalize a family to an ordered tuple of frozensets.

pytop.families.is_pairwise_disjoint_family(family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]]) → bool

Return whether every two distinct members are disjoint.

pytop.families.is_disjoint_family(family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]]) → bool

Alias for pairwise disjointness.

In this project, foundational notes use “disjoint family” in the usual pairwise sense, so the alias keeps the public surface linguistically simple.
