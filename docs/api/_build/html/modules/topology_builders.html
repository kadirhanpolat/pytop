Topology Builders

High-level topology builders for finite carriers.

Convenience constructors that hide frozenset plumbing and let users build standard topologies with clean mathematical syntax:

discrete_topology(1, 2, 3) make_topology({1, 2, 3}, {1}, {2, 3}) sierpinski_space()

pytop.topology_builders.make_topology(carrier: Iterable[Any], *open_sets: Iterable[Any]) → FiniteTopologicalSpace

Build a topology on carrier from the given explicit open sets.

∅ and X are added automatically — you only need to list the non-trivial open sets.

Parameters

carrier:

Elements of the underlying set X.

*open_sets:

Additional open sets (besides ∅ and X). Each may be any iterable of elements from carrier.

Examples

>>> sp = make_topology({0, 1, 2}, {0}, {1, 2})
>>> len(sp.topology)  # {∅, {0}, {1,2}, X} = 4
4
pytop.topology_builders.discrete_topology(*elements: Any) → FiniteTopologicalSpace

Return the discrete topology on the given elements.

Every subset is open.

Parameters

*elements:

Elements of the underlying set. Pass them as separate arguments: discrete_topology(1, 2, 3).

pytop.topology_builders.indiscrete_topology(*elements: Any) → FiniteTopologicalSpace

Return the indiscrete (trivial) topology on the given elements.

Only ∅ and X are open.

Parameters

*elements:

Elements of the underlying set.

pytop.topology_builders.cofinite_topology(*elements: Any) → FiniteTopologicalSpace

Return the cofinite topology on the given elements.

A set U ⊆ X is open iff U = ∅ or X U is finite. For finite X every subset satisfies this, so the result equals the discrete topology — but it is tagged as cofinite for educational distinction.

Parameters

*elements:

Elements of the underlying set.

pytop.topology_builders.sierpinski_space() → FiniteTopologicalSpace

Return the Sierpiński space: X = {0, 1}, τ = {∅, {1}, {0, 1}}.

The point 1 is open; the point 0 is not.

pytop.topology_builders.topology_from_basis(carrier: Iterable[Any], basis: Iterable[Iterable[Any]]) → FiniteTopologicalSpace

Construct the topology generated by a basis.

Delegates to pytop.subbases.generate_topology_from_basis().

Parameters

carrier:

Elements of the underlying set X.

basis:

A collection of subsets of X satisfying the basis axioms: (B1) covers X; (B2) intersections are locally refinable.

pytop.topology_builders.topology_from_subbasis(carrier: Iterable[Any], subbasis: Iterable[Iterable[Any]]) → FiniteTopologicalSpace

Construct the topology generated by a subbasis.

The topology is the smallest topology containing all subbasis members. Delegates to pytop.subbases.generate_topology_from_subbasis().

Parameters

carrier:

Elements of the underlying set X.

subbasis:

A collection of subsets of X. Finite intersections of these sets form a basis, and their unions form the topology.
