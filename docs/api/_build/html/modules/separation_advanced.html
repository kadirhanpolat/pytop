Separation Advanced

Advanced separation-axiom queries — T3 through T6 and Tychonoff multi-criterion.

This module contains: - Convenience wrappers for T3, T3.5 (Tychonoff), T4, T5, T6 and related properties - check_tychonoff() — multi-criterion Tychonoff decision procedure - tychonoff_characterization() — structured characterization report - separation_chain() — full T0–T6 ordered result dict

The shared engine, data constants, and basic queries (T0–T2.5) live in pytop.separation_basic. The legacy entry point pytop.separation re-exports everything from both modules.

pytop.separation_advanced.check_tychonoff(space: Any) → Result

Multi-criterion Tychonoff (T3.5 / completely regular Hausdorff) check.

Decision layers

  1. Explicit blocking tags (not_tychonoff, not_t3_5, not_completely_regular, not_t1) → false.

  2. Metric tag → true (every metric space is Tychonoff).

  3. Direct positive Tychonoff tags → true.

  4. T1 + completely_regular → true (characterisation theorem).

  5. T4 / normal + T1 → true (Urysohn’s Lemma); perfectly_normal is covered here too via _positive_tag_implies(tags, “t4”).

  6. Unknown otherwise.

pytop.separation_advanced.separation_chain(space: Any) → dict[str, Result]

Return the full separation hierarchy for the space.

The result is an ordered dict covering the standard chain T0 ≤ T1 ≤ T2 ≤ T2.5 ≤ T3 ≤ T3.5 ≤ T4 ≤ T5 ≤ T6, keyed by SEPARATION_CHAIN_ORDER. The Tychonoff entry uses check_tychonoff() (multi-criterion) rather than the basic tag lookup.
