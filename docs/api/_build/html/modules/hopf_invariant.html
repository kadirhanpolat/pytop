Hopf Invariant

Hopf invariant and related homotopy invariants (Phase 13.4).

The Hopf invariant of a map f: S^{2n-1} → S^n is an integer H(f) ∈ ℤ that measures the linking of the preimages of two regular values.

Algebraic definition (via mapping cone):

Let C(f) = S^n ∪_f e^{2n} be the mapping cone of f. H*(C(f); ℤ) has generators ι_n ∈ H^n and ι_{2n} ∈ H^{2n}. The cup product ι_n ∪ ι_n = H(f) · ι_{2n}. The Hopf invariant is the coefficient H(f) ∈ ℤ.

Key results

Hopf fibrations (maps of Hopf invariant 1):

η₁: S¹ → S¹ (n=1, H=1) — identity / degree 1 η₂: S³ → S² (n=2, H=1) — classical Hopf fibration, C(η₂) = CP² η₄: S⁷ → S⁴ (n=4, H=1) — quaternionic Hopf fibration, C(η₄) = HP² η₈: S¹⁵ → S⁸ (n=8, H=1) — octonionic Hopf fibration, C(η₈) = OP²

Adams theorem (1960): Maps of Hopf invariant 1 exist only for n = 1, 2, 4, 8.

Composition:

H(f ∘ g) = deg(g)² · H(f) (when g: S^{4n-1} → S^{2n-1}) H(f₁ + f₂) = H(f₁) + H(f₂) (in the stable range)

Linking number formula (n = 2):

For f: S³ → S², H(f) = lk(f^{-1}(p), f^{-1}(q)) for regular values p, q. This is the preimage-linking-number interpretation.

π₃(S²) = ℤ, generated by the Hopf fibration η₂ with H(η₂) = 1.

pytop.hopf_invariant.hopf_invariant_from_cup(cup_coefficient: int, n: int = 2) → int

Compute the Hopf invariant from the cup product structure in C(f).

Given H*(C(f); ℤ) = ℤ[ι_n, ι_{2n}]/(ι_n² = H(f)·ι_{2n}, ι_{2n}² = 0), the Hopf invariant H(f) is exactly the coefficient of ι_{2n} in ι_n ∪ ι_n.

Parameters

cup_coefficientint

The integer k such that ι_n ∪ ι_n = k · ι_{2n}.

nint

Target sphere dimension (unused; kept for API symmetry).

Returns

int

The Hopf invariant H(f) = cup_coefficient.

pytop.hopf_invariant.adams_hopf_invariant_one(n: int) → bool

Adams theorem: does there exist a map S^{2n-1} → S^n with H(f) = 1?

Returns True iff n ∈ {1, 2, 4, 8} (the only dimensions admitting maps of Hopf invariant one, by J.F. Adams 1960 using secondary cohomology operations / K-theory).

pytop.hopf_invariant.pi3_s2() → dict[str, object]

Return the computation of π₃(S²) = ℤ.

The third homotopy group of S² is generated by the Hopf fibration η₂: S³ → S² with Hopf invariant H(η₂) = 1.

Every element of π₃(S²) is of the form n·[η₂] with Hopf invariant n².

Wait: actually π₃(S²) ≅ ℤ and the Hopf invariant H: π₃(S²) → ℤ is an isomorphism. If [f] = n·[η₂] in π₃(S²), then H(f) = n.

pytop.hopf_invariant.hopf_invariant_composition(hopf_f: int, degree_g: int) → int

Hopf invariant of f ∘ g under suitable conditions.

If f: S^{2n-1} → S^n has Hopf invariant H(f) and g: S^{4n-1} → S^{2n-1} has degree deg(g), then H(f ∘ g) = deg(g)² · H(f).

Parameters

hopf_fint

Hopf invariant of f.

degree_gint

Degree of g (viewed as a self-map of a sphere).
