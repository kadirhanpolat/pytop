Borel Measures

Borel measures, Radon measures, regularity, and Riesz representation.

Key theorems implemented

  • A Borel measure on a topological space X is a measure defined on the Borel sigma-algebra B(X) generated by the open sets of X.

  • A Radon measure is a Borel measure that is locally finite (every point has a neighbourhood of finite measure) and inner regular (mu(A) = sup{mu(K) : K compact, K subset A}).

  • Riesz-Markov-Kakutani representation theorem: for a compact Hausdorff space X, every positive linear functional L on C(X) is represented by a unique regular Borel measure mu such that L(f) = integral f d-mu.

  • Regularity: a Borel measure is outer regular if mu(A) = inf{mu(U) : U open, A subset U} and inner regular if mu(A) = sup{mu(K) : K compact, K subset A}.

  • Haar measure: every locally compact topological group G admits a left-invariant Radon measure (Haar measure), unique up to a positive scalar multiple.

  • On compact groups, the Haar measure can be normalised to a probability measure.

  • The Lebesgue measure on R^n is translation-invariant, regular, non-atomic, and sigma-finite.

  • The Dirac measure delta_x is a probability measure supported at {x}; it is atomic and Radon.

  • The Cantor measure (Devil’s staircase measure) is supported on the Cantor set, is non-atomic, and is singular with respect to the Lebesgue measure.

  • On a locally compact Hausdorff space, every Radon measure is regular.

  • The counting measure on an uncountable discrete space is NOT locally finite, hence not Radon.

pytop.borel_measures.is_radon_measure(space: Any) → Result

Check whether the measure on space is a Radon measure.

A Radon measure on a topological space X is a Borel measure mu that is: (i) locally finite: every point x has a neighbourhood U with mu(U) < inf, (ii) inner regular: mu(A) = sup{mu(K) : K subset A, K compact} for all Borel A. Key facts: - On a compact Hausdorff space, every finite Borel measure is Radon. - On a locally compact Hausdorff space, every regular Borel measure is Radon. - The Lebesgue, Dirac, Haar, and Gaussian measures are all Radon. - The counting measure on an uncountable discrete space is NOT Radon

(fails local finiteness: every open set {x} has measure 1, but R is uncountable).

Decision layers

  1. Explicit Radon / Dirac / Lebesgue / Haar / Gaussian tag -> true.

  2. Compact Hausdorff space (finite Borel measures are Radon) -> true.

  3. Locally compact Hausdorff + regular Borel measure -> true.

  4. Counting measure on uncountable space -> false.

  5. Explicit not-Radon tag -> false.

  6. Unknown otherwise.

pytop.borel_measures.is_regular_measure(space: Any) → Result

Check whether the measure on space is regular (outer and inner regular).

A Borel measure mu on a topological space X is: - Outer regular: mu(A) = inf{mu(U) : U open, A subset U} for all Borel A. - Inner regular: mu(A) = sup{mu(K) : K compact, K subset A} for all Borel A. - Regular: both outer and inner regular. Key facts: - On a locally compact Hausdorff space, every Radon measure is regular. - The Lebesgue, Dirac, Haar, and Gaussian measures are all regular. - On a compact metric space, every Borel probability measure is regular

(Prokhorov / Ulam’s theorem).

Decision layers

  1. Explicit regular measure tags -> true.

  2. Radon measure on locally compact Hausdorff space -> true.

  3. Compact metric space (Ulam’s theorem) -> true.

  4. Explicit not-regular tag -> false.

  5. Unknown otherwise.

pytop.borel_measures.riesz_representation_applies(space: Any) → Result

Check whether the Riesz-Markov-Kakutani representation theorem applies.

The Riesz-Markov-Kakutani theorem: for a locally compact Hausdorff space X, every positive linear functional L on C_c(X) (continuous functions with compact support) is represented by a unique regular Borel measure mu such that L(f) = integral f d-mu for all f in C_c(X). For compact Hausdorff X: C_c(X) = C(X) and the dual C(X)* is exactly the space of signed regular Borel measures (by the full Riesz theorem). Key facts: - Compact Hausdorff: C(X)* cong M(X) (regular Borel measures). - Locally compact Hausdorff: C_c(X) functionals <- regular Borel measures. - Non-Hausdorff spaces: the theorem fails in general.

Decision layers

  1. Compact Hausdorff -> true (full Riesz theorem for C(X)*).

  2. Locally compact Hausdorff -> true (Riesz for C_c(X)).

  3. Non-Hausdorff / not T2 -> false.

  4. Infinite-dimensional without local compactness -> false.

  5. Unknown otherwise.

pytop.borel_measures.has_haar_measure(space: Any) → Result

Check whether space (as a topological group) admits a Haar measure.

Every locally compact topological group G admits a left Haar measure: a non-zero left-invariant Radon measure, unique up to a positive scalar multiple. Key facts: - Compact groups: normalised Haar measure is also a probability measure. - Discrete groups: Haar measure is the counting measure. - R^n (additive): Haar measure is the Lebesgue measure. - Infinite-dimensional topological groups (e.g., Hilbert spaces) generally

do NOT admit a Haar measure (no sigma-finite translation-invariant measure exists on infinite-dimensional Banach spaces).

Decision layers

  1. Explicit Haar measure tag -> true.

  2. Locally compact topological group (including compact groups) -> true.

  3. Discrete group (counting measure is Haar) -> true.

  4. Locally compact abelian group -> true.

  5. Infinite-dimensional Banach / Hilbert / Frechet (not locally compact) -> false.

  6. Unknown otherwise.

pytop.borel_measures.measure_support_is_compact(space: Any) → Result

Check whether the support of the measure is compact.

The support of a Borel measure mu on X is the smallest closed set F such that mu(X F) = 0. Equivalently, x is in supp(mu) iff every open neighbourhood of x has positive measure. Key facts: - Dirac measure delta_x: support = {x} (compact). - Measure on a compact space: support subset of compact space -> compact. - Lebesgue measure on R: support = R (not compact). - Gaussian measure on R^n: support = R^n (not compact). - Cantor measure: support = Cantor set C (compact). - Counting measure on Z: support = Z (not compact).

Decision layers

  1. Dirac measure (support = singleton) -> true.

  2. Measure on compact space (support subset compact) -> true.

  3. Compactly supported measure tag -> true.

  4. Cantor measure (support = Cantor set, compact) -> true.

  5. Lebesgue / Gaussian / Haar on R^n (support = R^n, not compact) -> false.

  6. Unknown otherwise.
