Filters

Finite filter and filter-base helpers for the topology book ecosystem.

The module is intentionally small: it gives Chapter 20 a testable finite bridge without pretending to solve arbitrary symbolic filter problems.

A filter base is a non-empty family closed under finite intersections in the sense that for any two base elements B1, B2 there exists B3 in the base with B3 ⊆ B1 ∩ B2. The generated filter consists of all supersets of base elements.

Filter convergence: a filter F converges to x if every neighborhood of x belongs to F (i.e. the neighborhood filter of x is a subset of F).

This module provides:

  • is_filter_base – check the filter-base axioms on a finite family

  • generated_filter – build the upward-closure filter from a base

  • is_filter – verify the filter axioms directly

  • neighborhood_filter_base – extract the open-neighborhood base at a point

  • filter_converges_to – check finite filter convergence

  • is_finer_filter – compare filters by refinement

  • filter_clusters_at – test whether a filter clusters at a point

  • filter_cluster_points – compute finite filter cluster/adherent points

  • analyze_filter – gather validity, convergence, and cluster data

exception pytop.filters.FilterError

Bases: ValueError

Raised when a finite filter request has a malformed input shape.

pytop.filters.is_filter_base(carrier: Iterable[Any], family: Iterable[Iterable[Any]]) → Result

Check whether family is a filter base on carrier.

A family B is a filter base when:

(B1) B is non-empty and does not contain the empty set. (B2) For every B1, B2 in B there exists B3 in B with B3 ⊆ B1 ∩ B2.

pytop.filters.generated_filter(carrier: Iterable[Any], base: Iterable[Iterable[Any]]) → Result

Build the filter generated by base on carrier.

The generated filter is {A ⊆ carrier : ∃ B ∈ base, B ⊆ A}. Returns the family as a sorted tuple of frozensets in the result value.

pytop.filters.is_filter(carrier: Iterable[Any], family: Iterable[Iterable[Any]]) → Result

Check whether family is a filter on carrier.

A filter F satisfies:

(F1) ∅ ∉ F. (F2) A, B ∈ F ⟹ A ∩ B ∈ F. (F3) A ∈ F and A ⊆ B ⊆ carrier ⟹ B ∈ F.

pytop.filters.neighborhood_filter_base(space: Any, point: Any) → Result

Return the open-neighborhood filter base at point in a finite space.

For a finite topological space the open sets that contain point form a filter base; their intersection (the minimal open neighborhood) is the smallest base element.

pytop.filters.filter_converges_to(space: Any, family: Iterable[Iterable[Any]], point: Any) → Result

Check whether a filter family converges to point in a finite space.

A filter F converges to x if every open neighborhood of x belongs to F. For finite spaces checking open neighborhoods is equivalent to checking all neighborhoods (since the minimal open neighborhood is the smallest neighborhood).

pytop.filters.is_finer_filter(carrier: Iterable[Any], finer: Iterable[Iterable[Any]], coarser: Iterable[Iterable[Any]]) → Result

Check whether one finite filter is finer than another.

With the usual order convention, finer is finer than coarser when every member of coarser also belongs to finer. Equivalently, coarser ⊆ finer as families of subsets of the carrier.
