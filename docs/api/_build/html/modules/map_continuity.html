Map Continuity

Pointwise continuity and criterion-based continuity checks for finite maps.

This module contains: - is_continuous_at_point() — neighborhood criterion at a single point - is_sequentially_continuous_at_point() — derived via first-countability - continuity_via_codomain_basis() — basis criterion (Chapter 06) - continuity_via_codomain_subbasis() — subbasis criterion (Chapter 06) - satisfies_closure_image_inclusion() — f(cl(A)) ⊆ cl(f(A)) check - initial_topology_from_maps() — coarsest topology making maps continuous

The core analysis engine, types, and private helpers live in pytop.finite_map_analysis. The legacy entry point pytop.maps re-exports everything from both modules.
