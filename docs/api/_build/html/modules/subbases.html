Subbases

Subbasis and basis helpers for explicit foundational computations.

The functions in this module stay deliberately finite and concrete. They are intended for Chapter 04 notebooks, worked examples, and lightweight internal validation around basis criteria, subbasis generation, and local-base checks.

exception pytop.subbases.BasisConstructionError

Bases: ValueError

Raised when a finite basis or subbasis construction is invalid.

pytop.subbases.is_basis_for_some_topology(carrier: Iterable[Any], basis_family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]]) → bool

Return whether basis_family is a basis for some topology on carrier.

The criterion used is the standard finite Chapter 04 criterion: the basis must cover the carrier, and intersections around any common point must be locally refinable by another basis element.

pytop.subbases.is_basis_for_topology(carrier_or_space: Iterable[Any] | TopologicalSpace, basis_family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]], topology: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]] | None = None) → bool

Return whether basis_family is a basis for the given topology.

The function accepts either (carrier, basis_family, topology=...) or a finite TopologicalSpace instance as the first argument.

pytop.subbases.finite_intersection_basis(carrier: Iterable[Any], subbasis_family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]]) → list[set[Any]]

Return the basis obtained from finite intersections of a subbasis.

The empty intersection is interpreted as the full carrier, so the output always contains X. Members are returned as a deduplicated list of sets.

pytop.subbases.generate_topology_from_basis(carrier: Iterable[Any], basis_family: Mapping[Any, Iterable[Any]] | Iterable[Iterable[Any]], *, description: str | None = None) → FiniteTopologicalSpace

Construct the finite topology generated by a basis.

The construction is exact: every union of basis members is enumerated explicitly. The result is returned as a FiniteTopologicalSpace.
