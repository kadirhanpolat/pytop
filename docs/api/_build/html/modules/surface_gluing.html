Surface Gluing

Polygon edge-identification helpers for the geometric topology bridge.

The module adds the GEO-07 surface-gluing layer. It parses small teaching edge words such as a b a^-1 b^-1 and records standard polygon models. It validates finite edge-pairing syntax and gives a convention-bound orientability heuristic, but it does not classify arbitrary quotient spaces.
