Continua

Continuum and Hilbert cube teaching profiles for the geometric topology bridge.

The helpers in this module deliberately remain conservative. They record standard examples and counterexamples for the route GEO-08 without claiming a general decision procedure for compactness, connectedness, metrizability, or continuum recognition.
