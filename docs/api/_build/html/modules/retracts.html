Retracts

Retracts, absolute retracts, and ANR teaching profiles.

This module is part of the geometric topology bridge for route GEO-08. It records conservative profile data about retracts, deformation retracts, absolute retracts (AR), and absolute neighbourhood retracts (ANR). The module is intentionally not a general decision procedure: unknown cases remain explicitly unknown, and registered examples carry their certification notes.
