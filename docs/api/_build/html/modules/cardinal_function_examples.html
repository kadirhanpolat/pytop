Cardinal Function Examples

Cardinal-function example catalog and comparison routes for Cilt IV (v0.1.72).

This module turns the v0.1.70 definition/comparison/example framework into a small reusable catalog of worked example patterns. The v0.1.72 extension adds comparison-exercise routes and notebook-route alignment records. It remains a durable source module, not a per-version release/report surface.
