Result

Structured result objects for exact, theorem-based, symbolic, and conditional output.

This module provides a small but usable reporting model for the core package. The aim is to keep the result contract explicit:

  • what was concluded,

  • how it was concluded,

  • under which assumptions,

  • and with what degree of mathematical strength.

pytop.result.merge_results(*results: Result) → Result

Merge several results conservatively.

The merged result keeps the weakest status among the inputs: unknown > conditional > false/true exact agreement. If incompatible truth values appear, the merged result becomes conditional.
