Tightness Network Profiles

Stable core-facing tightness/network profile registries.

This module promotes the most reusable parts of the former experimental tightness_network_profiles surface into the main pytop package. The profiles stay lightweight and chapter-aware so other layers can use a stable vocabulary without importing research-facing scaffolding.

The v0.1.77 extension makes the first Cilt V handoff explicit: each profile now records whether it belongs to the entry lane or the advanced lane. The entry lane is suitable for first reading, worksheet, and quick-check use; the advanced lane is reserved for warning-line or research-facing discussion.
