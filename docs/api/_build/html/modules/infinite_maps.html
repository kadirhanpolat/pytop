Infinite Maps

Structured map support for infinite and symbolic topological settings.

This module does not try to solve arbitrary continuity or quotient questions by inspection. Instead, it provides a small symbolic map layer that integrates with Result, explicit tags, and theorem-style implications.

class pytop.infinite_maps.SymbolicMap(domain: Any, codomain: Any, name: str = 'f', metadata: dict[str, ~typing.Any]=<factory>, tags: set[str] = <factory>)

Bases: object

A symbolic map between topological spaces.

The class is intentionally lightweight. The tags field encodes exact map facts already known about the map. The analysis helpers in this module then use these exact tags together with a few standard theorem implications.
