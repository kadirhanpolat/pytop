Cosmology Topology

Kozmoloji ve evren topolojisi ogretim profilleri (MAN-03).

Bu modul Adams & Franzosa Bolum 14.4–14.5 hattini davranissal kozmoloji simulasyonuna donusturmez. Amac, evrenin olasi global topolojik modellerini, yerel geometri sinyallerini ve gozlemsel ayirt etme yaklasimlarini denetlenebilir ogretim profilleri olarak paketlemektir.
