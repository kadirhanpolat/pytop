Maps

Unified map support — backward-compatible re-export.

The implementation is split across two modules:

  • pytop.finite_map_analysis — core types, property analysis engine, map-algebra helpers, builder functions, taxonomy utilities, all private helpers

  • pytop.map_continuity — pointwise continuity, basis/subbasis criteria, closure-image inclusion, initial topology construction

Import from those modules directly for a finer-grained dependency, or continue importing from this module for full backward compatibility.

A symbolic map between topological spaces.

The class is intentionally lightweight. The tags field encodes exact map facts already known about the map. The analysis helpers in this module then use these exact tags together with a few standard theorem implications.
