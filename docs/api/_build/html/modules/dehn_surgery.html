Dehn Surgery

First homology of Dehn surgery on a framed link (rational coefficients).

Given a link L = L₁ ∪ … ∪ Lₙ and a surgery coefficient pᵢ/qᵢ on each component, the surgered 3-manifold M has

H₁(M; ℤ) = coker(A), A_{ii} = pᵢ, A_{ij} = qᵢ · lk(Lᵢ, Lⱼ) (i ≠ j),

because H₁ of the link complement is freely generated by the meridians μ₁, …, μₙ, the longitude λᵢ = Σ_{j≠i} lk(Lᵢ, Lⱼ) μⱼ, and each surgery kills the curve pᵢ μᵢ + qᵢ λᵢ. The cokernel is read off the Smith normal form of A (reused from pytop.homology).

For integral surgery (qᵢ = 1) this is the symmetric linking matrix with the framings on the diagonal, and |H₁| = |det A| whenever A is non-singular.

This module also classifies lens spaces L(p, q) = (p/q)-surgery on the unknot: homeomorphism (q' ≡ ±q^{±1} mod p) and homotopy equivalence (q q' ≡ ±n² mod p) — both elementary, exact computations.

Pure Python, exact integer arithmetic, no dependencies.

Worked checks

  • (p/q) surgery on the unknot → L(p, q), H₁ = ℤ/p.

  • 0-surgery on the unknot → S¹ × S², H₁ = ℤ.

  • ±1-surgery on any knot → integral homology sphere, H₁ = 0.

  • 0-surgery on the Borromean rings → T³, H₁ = ℤ³.

  • E₈ plumbing (eight −2-framed unknots) → Poincaré homology sphere, H₁ = 0.

pytop.dehn_surgery.first_homology_of_surgery(coefficients: list[Any], linking_numbers: list[list[int]] | None = None) → FirstHomology

Return H₁ of the manifold obtained by surgery on a framed link.

Parameters

coefficients:

One surgery coefficient per component: an integer framing f (meaning f/1) or a pair (p, q) for rational p/q surgery.

linking_numbers:

The n × n matrix of pairwise linking numbers lk(Lᵢ, Lⱼ); the diagonal is ignored (the framing comes from coefficients). Defaults to all zeros (an unlink), so single-component surgery needs no matrix.

Returns

FirstHomology

pytop.dehn_surgery.first_homology_of_link_surgery(link: Any, coefficients: list[Any]) → FirstHomology

Return H₁ of surgery on a LinkDiagram.

The pairwise linking numbers are read from the diagram via pytop.knot_invariants.linking_matrix(); the diagonal (which that function sets to 0) is replaced by the supplied framings.

pytop.dehn_surgery.are_lens_spaces_homeomorphic(p: int, q1: int, q2: int) → bool

Whether L(p, q₁) ≅ L(p, q₂) (orientation-preservingly).

Reidemeister’s classification: a homeomorphism exists iff q₂ ≡ ±q₁^{±1} (mod p).

pytop.dehn_surgery.are_lens_spaces_homotopy_equivalent(p: int, q1: int, q2: int) → bool

Whether L(p, q₁) ≃ L(p, q₂) (homotopy equivalence).

Classification: L(p, q₁) ≃ L(p, q₂) iff q₁ q₂ ≡ ±n² (mod p) for some n. (L(7, 1) and L(7, 2) are homotopy equivalent but not homeomorphic — the standard example.)
