Predicate Relations

Relations defined by binary predicates.

Provides: - MathRelation — relation represented by a predicate on two MathSets - relation_on — homogeneous constructor: R ⊆ base × base - relation_between — heterogeneous constructor: R ⊆ domain × codomain - Structural tests — is_reflexive_on, is_symmetric_on, is_transitive_on, … - Pre-built constants — leq (≤), lt (<), geq (≥), gt (>), divides (∣)

Usage

>>> leq.holds(2, 5)
True
>>> leq.restrict_to([1, 2, 3])
{(1,1),(1,2),(1,3),(2,2),(2,3),(3,3)}
>>> divides.holds(3, 12)
True

class pytop.predicate_relations.MathRelation(name: str, domain: MathSet, codomain: MathSet, predicate: Callable[[Any, Any], bool], description: str = '')

Bases: object

A binary relation defined by a predicate over two :class:`MathSet`s.

The relation R ⊆ domain × codomain holds for (x, y) iff x ∈ domain, y ∈ codomain, and predicate(x, y) is True.

codomain:

Right-hand MathSet.

predicate:

predicate(x, y) returns True iff (x, y) ∈ R.

Optional longer description.

holds(x: Any, y: Any) → bool

Return True iff (x, y) ∈ R.

Checks that x ∈ domain and y ∈ codomain before applying the predicate; returns False (not an error) if either is out of range.

restrict_to(elements: Iterable[Any]) → set[tuple[Any, Any]]

Return {(x, y) : x, y ∈ elements and (x,y) ∈ R}.

restrict_between(domain_elements: Iterable[Any], codomain_elements: Iterable[Any]) → set[tuple[Any, Any]]

Return {(x, y) : x ∈ dom_elems, y ∈ cod_elems, (x,y) ∈ R}.

inverse() → MathRelation

Return the inverse relation R⁻¹ where (y,x) ∈ R⁻¹ iff (x,y) ∈ R.

compose(other: MathRelation) → MathRelation

Return other ∘ self: (x, z) holds iff ∃y: self(x,y) ∧ other(y,z).

The intermediate set is self.codomain (= other.domain). The predicate does a pointwise existential check: it requires a sample from the intermediate set to be provided via restrict_to or checked by the caller.

Note: the composed predicate is not closed-form — it raises PredicateRelationError when called unless a finite intermediate set can be derived from the domain sampler.

is_reflexive_on(elements: Iterable[Any]) → bool

Return True iff R is reflexive on elements: ∀x, (x,x) ∈ R.

is_symmetric_on(elements: Iterable[Any]) → bool

Return True iff R is symmetric on elements: (x,y) → (y,x).

is_transitive_on(elements: Iterable[Any]) → bool

Return True iff R is transitive on elements.

is_antisymmetric_on(elements: Iterable[Any]) → bool

Return True iff R is antisymmetric on elements: (x,y)∧(y,x) → x=y.

is_partial_order_on(elements: Iterable[Any]) → bool

Return True iff R is a partial order (reflexive + antisymmetric + transitive).

is_total_order_on(elements: Iterable[Any]) → bool

Return True iff R is a total order (partial order + totality).

is_equivalence_on(elements: Iterable[Any]) → bool

Return True iff R is an equivalence (reflexive + symmetric + transitive).
