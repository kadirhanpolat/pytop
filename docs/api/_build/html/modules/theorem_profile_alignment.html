Theorem Profile Alignment

Stable theorem-engine alignments for promoted profile registries.

This module gives the promoted profile families a theorem-facing vocabulary. The goal is intentionally modest: rather than turning profile registries into an automated proof search system, it records safe theorem alignments that the core theorem engine can expose with explicit profile metadata.
